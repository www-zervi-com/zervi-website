Image Requirements for Zervi WordPress Website

This document outlines the image assets needed for the Zervi WordPress website implementation.

Note: All images should be high-quality, professionally-taken photos that reflect Zervi's brand positioning as a premium automotive interior products manufacturer.

Logo and Brand Assets

  • Company Logo - Multiple formats (PNG with transparency, SVG preferred)
  • Favicon - 32x32 PNG or ICO file
  • Social Media Profile Image - 1200 x 630 pixels

Home Page Images

  • Hero Banner - Large header image showcasing premium products (1920 x 800 pixels)
  • Category Thumbnails - One image for each product category (800 x 600 pixels)
  • Company Overview - Images of facilities or manufacturing process (1200 x 800 pixels)
  • Client/Partner Logos - If applicable, logos of major clients or partners

Category Page Images

  • Automotive - Header image and product photos (various sizes)
  • Camping and Outdoor - Header image and product photos (various sizes)
  • Apparel - Header image and product photos (various sizes)
  • Design Studio - Header image and example work (various sizes)
  • Productions and Services - Header image and service illustrations (various sizes)
  • Technologies - Header image and technology visualizations (various sizes)

About Page Images

  • Company History - Timeline images (if applicable)
  • Team Photos - If featuring team members (800 x 600 pixels)
  • Facilities - Images of manufacturing facilities (1200 x 800 pixels)

Image Specifications

  • Format: JPEG for photographs, PNG for graphics with transparency
  • Resolution: 72-150 DPI (web standard)
  • Color Mode: RGB
  • File Size: Optimize for web, generally under 300KB per image
  • Naming Convention: Use descriptive names with hyphens (e.g., zervi-automotive-neoprene-cover.jpg)

Image Optimization Best Practices

  • Compress all images using tools like TinyPNG or ImageOptim
  • Consider using WebP format for better performance (with JPEG/PNG fallbacks)
  • Include descriptive alt text for accessibility and SEO
  • Use responsive images with appropriate srcset attributes
  • Consider lazy loading for images below the fold
WordPress Tip: Once the WordPress site is set up, we'll use the Smush plugin to automatically optimize all uploaded images.
